Hey There!

I'm Joshua Hinchliffe, A Computer Science Undergraduate and Musician

I enjoy understanding how computers work, and how to write better, more efficient code. I am currently working on a To-Do list app that allows you to nest lists, and an anagram solver that will solve multiple words.

For my A-Level project I wrote a program that, when given a file, would tell you the compression type which would give the best compression ratio, then apply that compression to it. It did so by analysing a very large amount of files and all the different compression types. When a request for best compression type was made, it would try to find an appropriate match and then look up the best compression type for it. Every file it encountered was added to the database and would make the program more accurate. It opperated on a client-server basis, with one central database, which clients would query and upload details to.

Another one of my passions is music. I play Piano (grade 8 standard), Acoustic Guitar, and Bass Guitar. On piano I enjoy playing ragtime, particularly Scott Joplin's work, and am working towards being able to play his complete piano works all the way through. On Acoustic Guitar I have a passion for fingerstyle, Where one attempts to fit as much of a song (Melody, Chords, Percussion etc) into just a guitar. I am particularly proud of my arrangements of Hallelujah and O Holy Night.
